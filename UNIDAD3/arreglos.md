<h2 align="center">📘 Que son los Arreglos</h2>

>Un **arreglo** o **array** es una estructura de datos que permite almacenar varios valores del mismo tipo bajo un solo nombre. Cada elemento del arreglo se identifica mediante una posición o índice, lo que facilita el acceso y la manipulación de la información. Los arreglos son muy útiles cuando se necesita trabajar con grandes cantidades de datos relacionados, como listas de números, calificaciones o registros [11].



>El uso de arreglos permite que los programas sean más eficientes y organizados, ya que evita la creación de muchas variables individuales. Además, ayudan a simplificar procesos repetitivos y a manejar datos de forma estructurada dentro de los algoritmos y programas.

---

<h2 align="center">🧩 Tipos de Arreglos</h2>

---

<h3 align="center">➡️ Arreglos Unidimensionales (Vectores)</h3>

>Los **arreglos unidimensionales**, también llamados **vectores**, almacenan los datos en una sola dimensión, es decir, en una lista ordenada.  
>Cada elemento ocupa una posición específica y se accede a él mediante un único índice.  
>Este tipo de arreglo es ideal para manejar conjuntos simples de datos relacionados entre sí [12].
