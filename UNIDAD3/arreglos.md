<h1 align="center">📦 Arreglos (Arrays)</h1>

---

<h2 align="center">📘 Que son los Arreglos</h2>

>Un **arreglo** o **array** es una estructura de datos que permite almacenar varios valores del mismo tipo bajo un solo nombre. Cada elemento del arreglo se identifica mediante una posición o índice, lo que facilita el acceso y la manipulación de la información. Los arreglos son muy útiles cuando se necesita trabajar con grandes cantidades de datos relacionados, como listas de números, calificaciones o registros [11]. El uso de arreglos permite que los programas sean más eficientes y organizados, ya que evita la creación de muchas variables individuales. Además, ayudan a simplificar procesos repetitivos y a manejar datos de forma estructurada dentro de los algoritmos y programas.

<h3 align="center">➡️ Arreglos Unidimensionales (Vectores)</h3>

>Los **arreglos unidimensionales**, también llamados **vectores**, almacenan los datos en una sola dimensión, es decir, en una lista ordenada. Cada elemento ocupa una posición específica y se accede a él mediante un único índice. Este tipo de arreglo es ideal para manejar conjuntos simples de datos relacionados entre sí [12].

<h3 align="center">↔️ Arreglos Bidimensionales (Matrices)</h3>

>Los arreglos bidimensionales, conocidos como matrices, organizan la información en dos dimensiones: filas y columnas. Su estructura es similar a una tabla, lo que permite representar datos de forma más estructurada. Las matrices se utilizan cuando la información requiere una organización más compleja, manteniendo una relación clara entre filas y columnas [12].

<h3 align="center">🧊  Cadenas de Caracteres (Strings)</h3>

>Una cadena se define como una sucesión lineal de caracteres destinada a la manipulación de texto. Su implementación varía drásticamente según el nivel del lenguaje:

>En C (Bajo Nivel): Se gestionan como arreglos estáticos de tipo char. Su longitud no se almacena explícitamente, sino que se delimita mediante un carácter centinela nulo al final.

>En Java (Alto Nivel): Se encapsulan dentro de la clase String. Esto trata al texto como un objeto inmutable, abstrayendo la gestión de memoria y facilitando su manipulación mediante métodos integrados."
